ColBERT — Late Interaction & MaxSim

Unlike DPR's single-vector comparison, ColBERT keeps per-token embeddings. The MaxSim operator finds the best match for each query token across all document tokens.

SColBERT(q, d) = ∑i maxj cos(qi, dj)

Each query token finds its best-matching document token. This preserves fine-grained matching that single-vector models lose.

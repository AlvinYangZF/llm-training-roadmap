Reciprocal Rank Fusion (RRF)

RRF merges multiple ranked lists without needing score calibration. Each document gets a fused score: RRF(d) = Σ 1/(k + rank(d)) where k=60. Simple, effective, and widely used in production.

RRF(d) = ∑r ∈ Rankers 1 / (k + rankr(d))    where k = 60

Contextual Retrieval & Late Chunking

Two modern techniques to improve chunk quality: Contextual Retrieval (Anthropic) prepends document context to each chunk, and Late Chunking (Jina) embeds the full document first, then splits embeddings.

Contextual Retrieval (Anthropic, 2024)

Standard Chunking

Which agreement? What document? Context is lost.

"This chunk is from Section 4 of the ACME SaaS Service Agreement, which covers term and termination."

Late Chunking (Jina, 2024)

Early Chunking (Standard)

Document → Split into chunks → Embed each chunk independently
Chunk 1 → Embed(Chunk 1) → v1
Chunk 2 → Embed(Chunk 2) → v2
Chunk 3 → Embed(Chunk 3) → v3
Each chunk embedded without cross-chunk context

Late Chunking

Document → Embed FULL doc → Split token embeddings by chunk boundaries
Full Doc → BERT(Full Doc) → [t1...tN]
Pool(t1...tk) → v1
Pool(tk+1...tm) → v2
Pool(tm+1...tN) → v3
Each token attends to full document via self-attention

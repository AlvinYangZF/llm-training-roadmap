CRAG — Corrective Retrieval

A T5-large evaluator (84.3% accuracy vs ChatGPT's 64.7%) scores retrieved docs and triggers 3 correction paths.

T5 Evaluator
→

INCORRECT→Discard docs → Web Search fallback → New docs
AMBIGUOUS→Combine refined docs + web results
FLARE

FLARE — Active Mid-Generation Retrieval

FLARE generates tokens with confidence scores. When confidence drops below θ, retrieval triggers mid-generation. The tentative text becomes the query.

LightRAG — Dual-Level Retrieval

LightRAG (2024) integrates graph structures with vector retrieval using dual-level retrieval: low-level for entity-specific queries, and high-level for topic/theme queries. This achieves +20% improvement on multi-hop questions vs vanilla RAG.

Vanilla RAG

Q: "How does Einstein's work connect to GPS?"
Chunk: "Einstein published relativity in 1905..."
Chunk: "GPS satellites orbit at 20,200 km..."
Disconnected chunks. Cannot trace multi-hop reasoning path.

LightRAG

Q: "How does Einstein's work connect to GPS?"
Einstein → developed → General Relativity
General Relativity → predicts → Time Dilation
Time Dilation → corrected_by → GPS Satellites
Graph traversal reveals the complete reasoning chain.

HippoRAG — Hippocampus-Inspired Retrieval

HippoRAG (NeurIPS 2024) maps the human memory system to RAG components: the Neocortex (LLM) extracts knowledge, Parahippocampal Regions (retrieval encoder) link entities, and the Hippocampus (knowledge graph) acts as associative memory.

1
Query: "What was the drug used to treat Alzheimer's in the Stanford study?"
2
Neocortex (LLM): Extract query entities → [Alzheimer's, Stanford, drug treatment]
3
Parahippocampal (Encoder): Link entities to KG nodes via similarity matching
4
Hippocampus (KG): Traverse graph → Stanford → conducted → Study X → used → Donepezil
5
Retrieve associated passages and generate answer with full context

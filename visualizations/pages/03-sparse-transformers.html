Sparse Transformers

O(N√N) Attention via Factorized Local + Strided Patterns

Section 1 Dense vs Sparse Attention Patterns

Dense attention connects every token to every other token, producing N² attention weights. Sparse Transformers factorize this into local + strided patterns, reducing computation to O(N√N).

Dense Attention
O(N²) = 256 connections
Sparse Attention

Section 2 Local Attention Pattern

Each token attends to itself and its √N nearest neighbors, forming a diagonal band across the attention matrix. This captures short-range dependencies efficiently.

Local attention band (√N = 4 wide)
No attention
Local pattern: each token sees √N nearest neighbors
Connectivity per token: √N = 4

Section 3 Strided Attention Pattern

Every √N-th position acts as a "summary" token that connects to all other tokens. These strided positions aggregate information from across the sequence, enabling long-range communication.

Strided attention columns (stride = √N = 4)
No attention
Strided pattern: attend to every √N-th position
"Summary" positions: 0, 4, 8, 12 aggregate global info

Section 4 Combined Coverage Animation
